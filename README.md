# hamesh

A local-first daily notes app — a per-day markdown editor with a calendar and inline time logging. All data is stored in the browser (localStorage).

## Stack

Vue 3 · Vite · [Milkdown](https://milkdown.dev/) (Crepe editor)

## Development

```sh
npm install      # install dependencies
npm run dev      # start the dev server (http://localhost:5173)
npm run build    # build for production → dist/
npm run preview  # preview the production build
```
